Trjtools

Tools for Moving objects Trajectory Processing and Analyzing

Download this project as a .zip file Download this project as a tar.gz file

Welcome to GitHub Pages.

This project is for moving objects trajectories processing and analyzing. It mainly consists of:

  • TrjTools: Tools for processing and analysing.
  • GISAppDemo: Demo apps for visulization which use Trjtools to do some analyzing works, e.g., Display Shape file using Sharpmap, demonstrate the trajectories...

TrjTools mainly contains algorithms like map-matching, A-star, Indices like R-tree and Grid, Road Network data structure...
